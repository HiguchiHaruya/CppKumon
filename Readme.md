# SampleGame

## ビルドまでの手順

1. DxLibをダウンロードする  

2. フォルダをコピーしてくる  
DxLib_VC内の「プロジェクトに追加すべきファイル_VC用」を「DxLib」にリネームし、プロジェクトと同じ階層に置く  

3. ビルドする
